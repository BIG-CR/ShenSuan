# 八字格局速断

三十年功力传统命理师 Claude Code Skill。输入公历生日，即刻排出八字、十神、调候用神、神煞，每法最少 400 字独立判词——医易同理、好坏都说、宜与不宜。八面全覆盖：格局定位、性格禀赋、事业路径、财富格局、人际图谱、感情画像、流年节点、行动指南。

## 使用

在 Claude Code 对话中直接输入出生信息即可触发：

```
2004年1月11日 早上10点35分 出生，福建仙游，男
```

Skill 自动激活，按四步工作流引导你完成：

1. **选分类** — 八字（十二法全出）/ 问事情（八法齐出）/ 推演环境（六法诊断）
2. **收信息** — 按分类逐项追问，信息不全不推进
3. **推演** — Python 精确计算日柱 + 硬性检查（真太阳时/节气边界/早夜子时）+ 全方法判词生成
4. **输出** — 每法最少 400 字独立判词，八面全覆盖，末尾附免责声明。用户档案自动写入 Claude auto-memory，下次对话自动回溯。

## 已知局限

- 日柱手工推算±1日，精确需 Python sxtwl 库
- 紫微斗数/七政四余/铁板神数等需专业排盘软件
- 演禽法仅支持十二生肖简化版
- 节气边界±1天可能跨月柱，需提示
- 真太阳时：西部偏远地区（经度<75°E）需手动修正时差
