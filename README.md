# 八字格局速断

三十年功力传统命理师 Claude Code Skill。输入公历生日，即刻排出八字、十神、调候用神、神煞，每法约 300 字独立判词——医易同理、好坏都说、宜与不宜。

## 安装

```bash
# 克隆到 Claude Code 项目的 .claude/skills 目录
git clone https://github.com/BIG-CR/神算-shensuan.git .claude/skills/神算-shensuan
```

或在 `.claude/skills/` 下创建 `神算-shensuan/` 目录，放入本仓库全部文件。

## 使用

在 Claude Code 对话中直接输入出生信息即可触发：

```
2004年1月11日 早上10点35分 出生，福建仙游，男
```

Skill 自动激活，按四步工作流引导你完成：
1. 收情报 + 直接排盘出核心四法判词（快速路径，三步到位），免责声明在输出末尾
2. 深度定制 — 用户要求时才展示 26 法概览 + 多选维度与方法
3. 追问缺信息 + 了解近况（合并一步），不填则用默认值
4. 确认生成判词（每种方法约 300 字）
